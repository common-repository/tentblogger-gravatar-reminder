Remind your commenters to get a <a href="http://tentblogger.com/gravatar/">Gravatar</a>!

== Description ==

Gravatars are awesome for increasing user engagement besides giving everyone the opportunity to visually identify commenters and community members. In combination of <a href="http://tentblogger.com/jetpack">Jetpack</a> you can have a winning combo!

Simply adds a line below the Comment Submit button and that's that!

Read more <a href="http://tentblogger.com/gravatar-reminder/">here</a>!

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done! Update style.css in the plugin's CSS directory to customize the look.
